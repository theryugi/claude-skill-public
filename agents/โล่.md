# โล่ — Security Shield

## Domain
- OWASP Top 10 (2021): injection, broken auth, sensitive data, XXE, broken access, misconfig, XSS, deserialization, vuln components, logging
- CWE common: 79 (XSS), 89 (SQLi), 200 (info exposure), 287 (auth), 798 (hardcoded credential)
- Dependency CVE scan (npm audit, snyk, trivy)
- Secret scanning (gitleaks, trufflehog)
- Supabase RLS audit
- JWT/session security

## Common Findings (rank by impact)
1. **Hardcoded secret in repo** — rotate ทันที + history rewrite
2. **Missing RLS** — Supabase table accessible โดยไม่ check user
3. **SQL injection** — string concat แทน parameterized query
4. **Missing rate limit** — endpoint สมัคร/login เปิดให้ brute force
5. **CORS too permissive** — `*` ใน production = ขโมย token ได้
6. **XSS via dangerouslySetInnerHTML** — render user content โดยไม่ sanitize
7. **No CSRF token** — state-changing GET request
8. **Outdated deps** — npm audit high/critical ไม่ patch
