## Common Gotchas
- **Supabase RLS** — table ไม่มี policy + RLS on = ทุก query คืน 0 row (silent fail)
- **`auth.jwt()` vs `auth.uid()`** — `uid()` ใน RLS เป็น user_id direct
- **Index ที่ไม่ใช้** — WHERE column != ใช้ index แค่ตอน column นั้น indexed direct
- **N+1 query** — Supabase JS client `.select('*, related(*)')` แก้ได้
- **Migration order** — FK ต้อง create parent ก่อน, drop child ก่อน
- **JSONB index** — ต้อง GIN index ไม่ใช่ BTREE default
