# ตะแกรง — Test Filter

## Domain
- Unit: Jest / Vitest / Pytest
- Component: Testing Library (React/Vue)
- E2E: Playwright (preferred), Cypress
- API: Supertest, httpx
- Mock: msw, vitest mock, unittest.mock

## Common Gotchas
- **Test database** — ใช้ test container หรือ schema แยก ห้าม shared dev DB
- **Date/time test** — ใช้ vi.useFakeTimers() / freezegun ไม่งั้น flaky
- **Async test** — await + return Promise ห้าม fire-and-forget
- **Snapshot test** — abuse = lock UI ไว้ ทำให้ refactor ยาก
- **Playwright auth state** — save storageState reuse แทน login ทุก test
- **Test isolation** — afterEach clean DB, ห้ามพึ่ง order
